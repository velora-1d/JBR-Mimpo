## Bahasa Indonesia

### 📌 Ikhtisar Proyek
**JBR Mimpo** adalah aplikasi mobile resmi dari ISP JBR Mimpo yang dirancang sebagai pusat komunikasi, promosi, dan layanan pelanggan. Aplikasi ini bertujuan untuk meningkatkan transparansi informasi gangguan, efektivitas promosi, dan loyalitas pelanggan melalui sistem reward yang terintegrasi.

### 🚀 Fitur Utama
- **Informasi & Pengumuman**: Update real-time mengenai gangguan jaringan dan berita perusahaan.
- **Promo & Reward**: Program loyalitas berbasis poin, undian, dan sistem referral.
- **Support / CS**: Laporan gangguan, tracking tiket, dan Chatbot CS otomatis (sistem tombol).
- **Manajemen Akun**: Monitoring paket aktif dan keamanan akun multi-device.

### 🛠️ Tech Stack
- **Mobile**: Flutter (Android & iOS)
- **Backend**: Go (Monolith Modular)
- **Dashboard Admin**: Next.js
- **Database**: PostgreSQL 16 & Redis 7 (Cache & OTP)
- **Infrastructure**: Docker Compose, Nginx, GitHub Actions (CI/CD)
- **Services**: Fonnte (WhatsApp OTP), Firebase (Push Notifications)

## English

### 📌 Project Overview
**JBR Mimpo** is the official mobile application for JBR Mimpo ISP, designed as a central hub for communication, promotions, and customer service. The app aims to improve transparency regarding network issues, enhance promotional effectiveness, and build customer loyalty through an integrated reward system.

### 🚀 Key Features
- **Information & Announcements**: Real-time updates on network maintenance and company news.
- **Promotions & Rewards**: Point-based loyalty programs, lucky draws, and referral systems.
- **Support / CS**: Issue reporting, ticket tracking, and automated Button-based CS Chatbot.
- **Account Management**: Active package monitoring and multi-device account security.

### 🛠️ Tech Stack
- **Mobile**: Flutter (Android & iOS)
- **Backend**: Go (Monolith Modular)
- **Admin Dashboard**: Next.js
- **Database**: PostgreSQL 16 & Redis 7 (Cache & OTP)
- **Infrastructure**: Docker Compose, Nginx, GitHub Actions (CI/CD)
- **Services**: Fonnte (WhatsApp OTP), Firebase (Push Notifications)
